Defined in the April Fools' RFC 2324, the Hyper Text Coffee Pot Control
Protocol: a teapot sent a `BREW` command must refuse and return
`418 I'm a teapot`. Extended by RFC 7168 (milk, via the Accept-Additions
header). In 2017, developers moved to strip 418 from real frameworks (Node.js,
Go) as a non-standard joke; a grassroots **"Save 418"** campaign preserved it.

The teapot as **conscientious objector** — an object that asserts its own
identity and refuses the task it was never built for. One of only two entries in
the collection where the teapot *names itself*; the other is
[[im-a-little-teapot]]. Belongs to the same refusal thread as the badger who
leaps off the fire ([[bunbuku-chagama]]) and the [[chocolate-teapot]] that
works despite being declared useless.
